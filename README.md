# e-plantShopping

A Vite + React storefront for browsing indoor plants by category and adding favorites to a cart. The app opens with a landing page and About Us section before sliding into a product list where users can explore curated plant collections, view details and costs, and manage selected items through Redux-powered cart interactions.

## Features
- Landing page with hero messaging and "Get Started" flow into the catalogue.
- Product listings grouped by plant benefits (air purifying, aromatic, medicinal, etc.) with imagery and pricing.
- Shopping cart state managed with Redux Toolkit, supporting add/remove and quantity updates.
- Responsive styling tailored for the plant nursery theme.

## Tech Stack
- React 18 with Vite tooling.
- Redux Toolkit and React Redux for state management.
- CSS modules scoped per feature.
